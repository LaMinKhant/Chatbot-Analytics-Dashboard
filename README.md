# Chatbot Analytics Dashboard 📊

## [English]

This repository showcases the data analysis of user interactions from my portfolio chatbot, visualized using MongoDB Atlas Charts. It highlights my ability to manage data pipelines and extract meaningful insights from raw conversation logs.

### 🔗 Live Links

* **Portfolio Website:** [https://laminkhant-portfolio.vercel.app/](https://laminkhant-portfolio.vercel.app/)
* **Live Dashboard:** [View Real-time Chatbot Analytics](https://charts.mongodb.com/charts-atlas-orange-pocket-cmqciux/public/dashboards/82eddee9-a7b2-4c60-a1a4-41026bfc2094)

### 💡 Overview

I track and analyze user interactions from my portfolio website to understand user behavior. Key metrics include:

* **Total Traffic:** Total message volume.
* **User Engagement:** Average messages per session.
* **Peak Hours:** High-activity time periods.
* **Language/Region Distribution:** Geographic and linguistic patterns.

## [မြန်မာဘာသာ]

ဒီ Repository သည် ကျွန်ုပ်၏ Portfolio Chatbot မှရရှိသော အချက်အလက်များကို MongoDB Atlas Charts ကိုအသုံးပြု၍ ခွဲခြမ်းစိတ်ဖြာထားသော (Analysis) ပရောဂျက်တစ်ခု ဖြစ်ပါသည်။ ဤသည်မှာ ကျွန်ုပ်၏ Data Engineering နှင့် Analytics စွမ်းရည်များကို ဖော်ပြပေးပါသည်။

### 🔗 Live Links

* **Portfolio Website:** [https://laminkhant-portfolio.vercel.app/](https://laminkhant-portfolio.vercel.app/)
* **Live Dashboard:** [Real-time Chatbot Analytics ကို ကြည့်ရှုရန်](https://charts.mongodb.com/charts-atlas-orange-pocket-cmqciux/public/dashboards/82eddee9-a7b2-4c60-a1a4-41026bfc2094)

### 💡 Overview

ကျွန်ုပ်၏ Portfolio ဝဘ်ဆိုက်ရှိ Chatbot မှတဆင့် ဝင်ရောက်လာသော မေးခွန်းများနှင့် အပြန်အလှန်ပြောဆိုမှုများကို စုဆောင်းကာ အောက်ပါအချက်အလက်များကို စဉ်ဆက်မပြတ် စောင့်ကြည့်လေ့လာလျက်ရှိပါသည်။

* **Total Traffic:** စုစုပေါင်း မက်ဆေ့ချ် အရေအတွက်။
* **User Engagement:** တစ်ကြိမ်ချင်းစီ၏ ပျမ်းမျှ မက်ဆေ့ချ် အရေအတွက်။
* **Peak Hours:** အသုံးပြုသူများ အများဆုံး အပြန်အလှန်ပြောဆိုသည့် အချိန်ကာလ။
* **Language/Region Distribution:** အသုံးပြုသည့် ဘာသာစကားနှင့် တည်နေရာအလိုက် ခွဲခြားမှု။
